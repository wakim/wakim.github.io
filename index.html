    Resumo

    Graduado em Bacharelado em Ciências da Computação pela UFRJ.

    Sou desenvolvedor Full Stack, especialmente nas áreas Web e Mobile. Motivado por novas técnologias, ferramentas e práticas, voltadas para melhoria no desenvolvimento, aumento de produtividade e inovação em soluções.

    Comecei na área de desenvolvimento WEB com manutenção e desenvolvimento de aplicações Web utilizando parte do JavaEE (JSP, Servlets e JDBC) no back-end. HTML, JavaScript e CSS no front-end.

    Minha segunda experiência em Web foi com JavaEE, utilizando EJB, JSF e JPA (Hibernate) no back-end. HTML, JavaScript (MVVM usando KnockoutJS) e CSS no front-end.

    Meu interesse por Mobile surgiu motivado pelo Projeto Final, serviu como uma oportunidade para explorar o mundo Android. Possuo um app publicado e alguns em desenvolvimento (nas horas livres), originados de estudos utilizando sempre os padrões da plataforma.
